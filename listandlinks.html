Steps to creating a new webpage

1) Open terminal 2)go to your repsitory by typing "cd" and the name of your folder 3) Create an index by typing "gedit" and the name of the index you wanted to create 4) Paste all of the content in the template.html 5) Add information to the document 6) Create a main.css by typing "gedit" and "main.css" 7) Go back to the index document and add a link which links the index to the main.css () 8) Save the index document by pressing shift and 8 (make sure there is no asterisk) 9) Type "git add" and the index name 10) Type git commit -m " " and add a message between the quotation marks. 11) Type "git push" to save the work. 12) Type the code (python -m SimpleHTTPServer) into the terminal 13) Open a new tab and type localhost:8000 14) Decorate the index in the main.css 15) Add the declaration and properties to style the document 16) Save the changes you main in the (main.css) document (by pressing shift and 8) 17) Go to the localhost"8000 tab and refresh to see if the changes have been made to the document. 18) Type "git add main.css" 19) Type git commit -m " " and add a message between the quotation marks.
